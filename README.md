# TypeScript API

## Getting started
Clone this repo:
```
git clone https://github.com/raphaellima8/ts-node-api.git && cd ts-node-api
```

Install dependencies:
```
npm i
```

Populate MongoDB
```
mongoimport --db martan --collection products --file seed.json --jsonArray
```

Run server in dev mode
```
npm start:dev
```
